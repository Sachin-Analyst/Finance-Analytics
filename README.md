# Finance-Analytics
This project focuses on providing actionable financial insights for a global consumer electronics business by transforming raw sales and cost data into a structured Profit and Loss (P&amp;L) framework. By centralizing data from 2019 to 2021, the analysis tracks key performance indicators across 50 markets to support data driven decision making.

## Project Description

This project establishes a comprehensive financial intelligence framework for AtliQ Hardwares, transitioning the organization from manual data tracking to a centralized, automated analysis of global profitability. By utilizing Advanced Excel and Pivot Tables, the analysis evaluates business health across multiple fiscal years (2019–2021) to identify critical trends in revenue growth and margin fluctuations.

## Key features
- Global Financial Performance Tracking
- P&L Reports by Fiscal Year
- Granular Monthly and Quarterly Analysis
- Market-Specific Profitability Insights
